## 🛠 How to Use

### 🧑‍💻 Manual Mode

1. Select your preferred region
2. Click **Convert**

### ⚙️ Automatic Mode

1. Select your preferred region
2. Click **Auto**
   → Any new or modified files will be auto-converted to the selected region
